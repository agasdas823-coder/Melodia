# Melodia

Melodia is a premium, distraction-free music streaming platform designed for discerning listeners. It combines intelligent cloud streaming with a pure, sleek, and highly responsive user interface to deliver an unparalleled listening experience.

## ✨ Features

- **High-Fidelity Audio Streaming**: Seamless playback powered by a robust Node.js backend.
- **YouTube Integration**: Search and stream high-quality audio directly from YouTube sources with custom API integrations.
- **Instant Lyrics**: Deep integration with Genius API to fetch and display synchronized lyrics for your favorite tracks instantly.
- **Pre-fetching Engine**: Intelligently buffers audio tracks and lyrics in the background upon hovering, allowing zero-latency playback.
- **Modern User Interface**: A beautifully crafted UI using React, Tailwind CSS, and Framer Motion, featuring glassmorphism, dynamic color themes, and subtle micro-animations.
- **Custom Playlists**: Create, manage, and share your personal music library with full playlist control.
- **Google OAuth Integration**: Secure, one-click sign-up and login flow powered by Google Identity Services.
- **Responsive Design**: Flawless experience on both desktop and mobile, with dedicated adaptive layouts.

## 🚀 Built With

- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Lucide React, Howler.js
- **Backend**: Node.js, Express, Youtube-dl/play-dl integrations, JWT Auth
- **AI-Assisted Development**: Built with the power of **Antigravity** for rapid prototyping, seamless full-stack integration, and next-level vibe coding.

## 💡 What can it be used for?

Melodia serves as an excellent foundation for:
- A personalized cloud music player for personal use.
- A template for building media streaming services.
- Demonstrating advanced React patterns (e.g., streaming states, audio context management, intelligent pre-loading).
- A showcase of modern web design aesthetics and complex UI animations.

## 🛠 Setup & Installation

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   # Install client dependencies
   npm install

   # Install server dependencies
   cd server
   npm install
   ```
3. **Environment Setup**:
   Create a `.env` file in the root directory based on the provided `.env.example`.
4. **Run the Application**:
   ```bash
   npm run dev
   ```
   This will start both the Vite client and the Express backend concurrently.
